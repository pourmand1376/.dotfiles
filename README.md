# dotfiles

I managed my dotfiles using `stow`. [This](https://www.youtube.com/watch?v=FHuwzbpTTo0) youtube video explains it very well. Also [this](https://www.jakewiesler.com/) website is good.

## Stow

For using config files, use:

```bash
stow folder_name
```
